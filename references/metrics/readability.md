# 可读性评分

代码可读性评分。

## 评分维度

- 超长函数数量（>100 行）
- 超长文件数量（>500 行）
- 深层嵌套数量（>3 层）
- 超长表达式数量
- 注释缺失的关键逻辑

## 评分公式

```
分数 = 100 - (超长函数 × 8) - (超长文件 × 5) - (深层嵌套 × 6) - (超长表达式 × 3) - (无注释关键逻辑 × 5)
```

最低 0 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 一目了然 |
| 75+ | ★★★★☆ | 轻松阅读 |
| 60+ | ★★★☆☆ | 需要耐心 |
| 40+ | ★★☆☆☆ | 阅读困难 |
| 0+  | ★☆☆☆☆ | 天书 |
