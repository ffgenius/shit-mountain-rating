# 命名规范评分

命名规范评分。

## 评分维度

- 拼音命名数量
- 无意义变量名数量（a, b, tmp, data 等）
- 过度缩写数量
- 命名风格不一致数量（camelCase / snake_case 混用）
- 同一概念不同命名数量

## 评分公式

```
分数 = 100 - (拼音命名 × 8) - (无意义变量 × 6) - (过度缩写 × 4) - (风格不一致 × 5) - (同义异名 × 5)
```

最低 0 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 命名规范 |
| 75+ | ★★★★☆ | 命名清晰 |
| 60+ | ★★★☆☆ | 命名有瑕疵 |
| 40+ | ★★☆☆☆ | 命名混乱 |
| 0+  | ★☆☆☆☆ | 全凭猜 |
