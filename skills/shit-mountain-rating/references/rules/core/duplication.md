# 重复度评分规则

分析代码重复度。

## 检测项

- 重复逻辑块（>6 行相同或高度相似代码）
- 重复 SQL（相同查询出现在多个位置）
- 重复正则表达式
- 相似函数（仅参数不同，相似度 >80%）
- 复制粘贴代码块（注释、变量名完全一致的连续代码）
- 重复字符串/常量字面量（同一含义的字符串在多处独立定义）

## 阈值定义

| 指标 | 警告 | 严重 |
|------|------|------|
| 重复逻辑块 | 2 处 | 3 处及以上 |
| 重复 SQL | 2 处 | 3 处及以上 |
| 相似函数 | 2 个 | 3 个及以上 |
| 复制粘贴块 | 1 处 | 2 处及以上 |
| 重复字符串 | 3 处 | 5 处及以上 |

## 评分公式

```
分数 = 100 - (重复逻辑块严重 × 15) - (重复SQL严重 × 10) - (相似函数严重 × 8) - (复制粘贴块严重 × 8) - (重复字符串严重 × 3) - (警告 × 2)
```

最低 0 分。重复业务逻辑扣分最重——同样的 bug 要修 N 次。

## 检测方式

- 代码块滑动窗口相似度对比（最小匹配 6 行）
- SQL 字符串提取和去重
- AST 结构相似度对比（忽略变量名和字面量）
- 字符串字面量收集和频率统计
