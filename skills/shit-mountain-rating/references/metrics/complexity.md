# 复杂度评分

复杂度维度评分。

## 评分维度

- 圈复杂度
- 认知复杂度
- 嵌套深度
- 函数/文件大小

## 评分公式

```
Score = 100 - (严重超限函数 × 10) - (超长文件 × 8) - (高嵌套 × 5)
```

最低 0 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 简洁优雅 |
| 75+ | ★★★★☆ | 较低复杂度 |
| 60+ | ★★★☆☆ | 复杂度适中 |
| 40+ | ★★☆☆☆ | 过于复杂 |
| 0+  | ★☆☆☆☆ | 复杂度灾难 |
