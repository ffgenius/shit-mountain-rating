# 架构评分

架构维度评分。

## 评分维度

- 分层清晰度
- 模块职责划分
- God Object 数量
- 层违规数量

## 评分公式

```
Score = 100 - (God Object × 15) - (Layer Violation × 10) - (职责不清模块 × 5)
```

最低 0 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 架构优秀 |
| 75+ | ★★★★☆ | 架构良好 |
| 60+ | ★★★☆☆ | 架构一般 |
| 40+ | ★★☆☆☆ | 架构较差 |
| 0+  | ★☆☆☆☆ | 架构灾难 |
