# 注释标记评分

代码注释标记统计与趋势评分。

## 评分维度

- TODO 数量
- FIXME 数量
- HACK 数量
- XXX 数量
- @deprecated 仍在使用数量
- 标记增长趋势（与上次分析对比）

## 评分公式

```
分数 = 100 - (FIXME × 8) - (HACK × 6) - (XXX × 4) - (TODO × 2) - (@deprecated仍用 × 5) - (增长趋势 × 10)
```

最低 0 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 标记可控 |
| 75+ | ★★★★☆ | 标记较少 |
| 60+ | ★★★☆☆ | 标记堆积 |
| 40+ | ★★☆☆☆ | 标记泛滥 |
| 0+  | ★☆☆☆☆ | 放弃治疗 |
