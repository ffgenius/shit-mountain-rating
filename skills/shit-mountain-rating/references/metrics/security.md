# 安全评分

安全维度评分。

## 评分维度

- SQL 注入
- 命令注入
- 硬编码密钥
- XSS / CSRF
- Eval 使用

## 评分公式

```
Score = 100 - (注入漏洞 × 25) - (硬编码密钥 × 20) - (XSS × 10) - (CSRF × 5)
```

最低 0 分。每个 SQL/Command 注入发现直接扣 25 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 安全 |
| 75+ | ★★★★☆ | 较安全 |
| 60+ | ★★★☆☆ | 有隐患 |
| 40+ | ★★☆☆☆ | 不安全 |
| 0+  | ★☆☆☆☆ | 高风险 |
