# 性能评分

性能维度评分。

## 评分维度

- N+1 查询数量
- 阻塞 I/O 数量
- 内存泄漏
- 重复计算

## 评分公式

```
Score = 100 - (N+1 × 15) - (Blocking IO × 10) - (内存泄漏 × 20) - (重复计算 × 5)
```

最低 0 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 性能优秀 |
| 75+ | ★★★★☆ | 性能良好 |
| 60+ | ★★★☆☆ | 性能一般 |
| 40+ | ★★☆☆☆ | 存在瓶颈 |
| 0+  | ★☆☆☆☆ | 性能灾难 |
