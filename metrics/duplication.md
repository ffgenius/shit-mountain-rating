# 重复度评分

代码重复度评分。

## 评分维度

- 重复业务逻辑块数量
- 重复 SQL 片段数量
- 相似函数数量（相似度 >80%）
- 复制粘贴代码块数量
- 重复字符串/正则数量

## 评分公式

```
分数 = 100 - (重复业务逻辑 × 15) - (重复SQL × 10) - (相似函数 × 8) - (复制粘贴块 × 5) - (重复字符串 × 2)
```

最低 0 分。重复业务逻辑扣分最重——相同的 bug 要修 N 次。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | DRY |
| 75+ | ★★★★☆ | 轻微重复 |
| 60+ | ★★★☆☆ | 重复明显 |
| 40+ | ★★☆☆☆ | 大量重复 |
| 0+  | ★☆☆☆☆ | 复制粘贴工程 |
