# 测试评分

测试维度评分。

## 评分维度

- 测试覆盖率
- Mock 质量
- 全局状态数量
- 可测试性

## 评分公式

```
Score = 100 - (无测试模块 × 20) - (全局状态 × 10) - (不可测试代码 × 15)
```

最低 0 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 测试完善 |
| 75+ | ★★★★☆ | 测试良好 |
| 60+ | ★★★☆☆ | 有测试 |
| 40+ | ★★☆☆☆ | 测试不足 |
| 0+  | ★☆☆☆☆ | 无测试 |
