# 可维护性评分

可维护性维度评分。

## 评分维度

- 模块独立性
- 修改影响范围
- 可扩展性
- 文档完整度

## 评分公式

```
Score = 100 - (高耦合模块 × 10) - (不可测试模块 × 10) - (修改影响 × 5)
```

最低 0 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 高度可维护 |
| 75+ | ★★★★☆ | 较易维护 |
| 60+ | ★★★☆☆ | 可维护 |
| 40+ | ★★☆☆☆ | 维护困难 |
| 0+  | ★☆☆☆☆ | 无法维护 |
