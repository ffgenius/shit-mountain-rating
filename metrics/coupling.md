# 耦合度评分

模块间耦合度评分。

## 评分维度

- 循环依赖数量
- 高 Import 模块数（>15 个依赖）
- 跨层依赖数量
- 依赖密度（模块被依赖数/依赖数）

## 评分公式

```
分数 = 100 - (循环依赖 × 20) - (高Import模块 × 8) - (跨层依赖 × 10) - (依赖密度过高模块 × 5)
```

最低 0 分。循环依赖每个直接扣 20 分——这是架构级硬伤。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 松耦合 |
| 75+ | ★★★★☆ | 耦合合理 |
| 60+ | ★★★☆☆ | 耦合偏紧 |
| 40+ | ★★☆☆☆ | 耦合严重 |
| 0+  | ★☆☆☆☆ | 意大利面 |
