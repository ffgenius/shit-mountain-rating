# 技术债评分

技术债综合评分。

## 评分维度

> 注：FIXME/TODO/HACK/XXX 标记计数归 [注释标记评分](comments.md)，本维度聚焦死代码和废弃接口。

- 死代码文件数（从入口不可达）
- 未使用的函数/变量数量
- 仍在使用的 @deprecated 接口
- 注释掉的大段代码块

## 评分公式

```
分数 = 100 - (死代码文件 × 12) - (未使用函数 × 6) - (@deprecated仍用 × 8) - (注释代码块 × 4)
```

最低 0 分。

## 星级映射

| 分数 | 星级 | 评级 |
|------|------|------|
| 90+ | ★★★★★ | 无技术债 |
| 75+ | ★★★★☆ | 技术债可控 |
| 60+ | ★★★☆☆ | 技术债积累 |
| 40+ | ★★☆☆☆ | 技术债严重 |
| 0+  | ★☆☆☆☆ | 技术债灾难 |
